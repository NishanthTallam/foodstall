# 🥤 Sweet Cravings — Back Benchers Stall

> A fully interactive, single-file digital menu website for **Sweet Cravings**, a milkshake & dessert stall.

---

## 📌 About

**Sweet Cravings** is a fun, youth-oriented stall serving thick milkshakes, refreshing lassis, Indian sweet desserts, and combo deals. This website serves as a **digital menu board** designed to be shared, displayed on a screen, or printed as a QR-linked page.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Warm & Cozy Theme** | Deep caramel + cream gradient background throughout |
| 🖼️ **Real Food Photos** | All 14 menu items use actual uploaded product images |
| 🌅 **Transparent BG Shakes** | 8 milkshake images float subtly in the background at 11% opacity |
| 🔥 **Popular Banner** | Auto-scrolling carousel of the 6 most popular milkshakes at the top |
| ⚡ **Deal of the Day** | Highlighted combo section inside the hero with animated badges |
| 📱 **Fully Responsive** | Works on mobile, tablet, and desktop screens |
| 🎬 **Smooth Animations** | Fade-in cards, bounce hero emoji, pulsing deal badges, sparkle effects |

## 🛠️ Tech Stack

- **Pure HTML5** — single self-contained file, no frameworks
- **CSS3** — custom properties, gradients, keyframe animations, backdrop-filter
- **Vanilla JavaScript** — banner scroll, background image fade-in
- **Google Fonts** — `Pacifico` (display) + `Nunito` (body)
- **Base64 Images** — all photos embedded directly, no external image hosting needed

## 🚀 How to Use

### Option 1 — Open Directly
Double-click `sweet-cravings.html` to open in any browser. No server needed.

### Option 2 — Share via QR Code
1. Upload `sweet-cravings.html` to any free hosting (GitHub Pages, Netlify, etc.)
2. Generate a QR code pointing to the URL
3. Print and display at the stall

### Option 3 — Display on Screen
Open the file full-screen on a tablet or monitor at your stall as a live digital menu.

## 🎨 Design Highlights

- **Color Palette:**
  - Background: `#FFF3DC → #FFDAA0` (warm cream-caramel gradient)
  - Primary: `#E8943A` (caramel orange)
  - Dark accent: `#5C2D0A` (deep chocolate brown)
  - Gold: `#F5C842`
  - Pink badge: `#F2607A`

- **Fonts:**
  - Headings: `Pacifico` (cursive, warm & fun)
  - Body: `Nunito` (rounded, friendly, readable)

- **Background Effect:**
  8 milkshake images (Kitkat, Mango, Chocolate, Strawberry, Oreo, Butterscotch, Rose Milk, Dark Fantasy) are placed left and right alternately across the full page at **11% opacity** with a soft blur, creating a dreamy layered effect.

---

## 📸 Images Used

All images were provided by the stall owner and embedded as Base64 PNG data directly into the HTML file. No external image links are used, making the file **fully offline-capable**.
